# HeaderSpy

A lightweight Python tool to check for missing security headers on websites.  
Useful for ethical hackers and bug bounty beginners.

## Features

- Checks for:
  - Content-Security-Policy
  - Strict-Transport-Security
  - X-Content-Type-Options
  - X-Frame-Options
  - Referrer-Policy
  - Permissions-Policy
